# 🎉 Party Preparation Agent

A Party Preparation AI Agent built using

- SmolAgents
- Gemini 2.5 Flash
- DuckDuckGo Search
- Gradio

---

## Features

✅ Search music

✅ Suggest Menu

✅ Suggest Dress Code

✅ Find Best Catering Service

✅ Calculate Party Preparation Time

---

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```

Paste your Gemini API key.

Start chatting.
